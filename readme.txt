Це заготовка бібліотеки для JS, у класичному стилі. 

Тут реалізовані наступні компоненти: 
    - Аккардіон
    - Кнопки
    - Карточти
    - Дропдаун меню
    - Модальні вікна
    - Слайдер
    - Таби

Реалізовано базові стилі через Класси: 
    - Центрування текстів 
    - Центрування блоків 
    - Шрифти 
    - Марджині
    - Падінги 
    - Розміри блоків
